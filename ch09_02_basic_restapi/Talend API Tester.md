# Talend API Tester

REST API 테스트 Tool



### (1) REST API 테스트 Tool

1. Talend API Tester 확장 프로그램 추가 : chrome 웹 스토어

2. **POST Test** : Create
   - Talend API Tester에서 <u>method, URI 및 body</u> 입력 후 'Send'
   - URI 접속으로 작동 결과 확인 불가 -> 웹 앱을 만들어야 테스트 가능

3. **GET Test** : Read

   - Talend API Tester에서 <u>method, URI</u> 입력 후 'Send'

   - URI 접속으로 작동 결과 확인 가능



### (1.1) Response 결과

- **200 OK** : 정상 작동
- 그 외 : 오류
